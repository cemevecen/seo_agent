# Claude Code — Proje Kuralları

## Git Branş Kuralı

**Tüm değişiklikler doğrudan `main` branch'e commit edilip push edilecek.**

- Feature branch, geçici branch, worktree KULLANMA.
- Her zaman `git checkout main` ile main'de olduğunu doğrula.
- Commit sonrası `git push origin main` ile main'e push et.
- Pull request AÇMA (kullanıcı açıkça istemediği sürece).
